#Murajaah Shell (murosh)

Simple CLI for tracking and planning murojaah from terminal

##Installation

```bash
git clone https://github.com/ilhamsadewarsa/murosh
cd murosh
pip install -e
```
## Available Command
murosh 
murosh add
murosh add [group] [name]
murosh remove [name]
murosh show [group]
murosh check [name]
murosh read [name]
murosh read random [number of part]
